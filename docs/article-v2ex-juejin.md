# Slidememos：浏览器侧边栏的便利贴

开源了一个浏览器侧边栏笔记工具 [Slidememos](https://github.com/yourusername/slidememos)。

定位很简单：**浏览器里的便利贴**，随时记录，随手查看，不跳出当前页面。

## 解决什么场景

**技术人员**
- 存 API Key、服务器地址、数据库连接字符串
- 常用命令行片段
- 项目配置信息

**写作者**
- 刷到好素材链接，顺手保存
- 灵感闪现，立即记录
- 文章大纲草稿

**日常办公**
- 临时待办事项（文本形式）
- 剪贴板历史
- 会议记录草稿

## 核心设计

**1. 卡片可折叠**

有些内容不需要一直看到，比如 API Key 或敏感配置。点击折叠按钮，卡片收起只显示标题，保护隐私同时保持界面整洁。

**2. Markdown 格式支持**

支持 Markdown 语法，标题、列表、代码块都能正确渲染显示。点击卡片进入编辑状态查看原始文本。

**3. 文本选中快捷操作**

编辑状态下选中任意文本，右上角弹出复制/剪切按钮。比右键菜单快，比快捷键直观。

**4. 链接卡片一键打开**

保存的链接卡片，点击左上角 🔗 图标直接在新标签页打开，不用展开复制粘贴。

## 数据存储

所有内容存在浏览器本地，通过 Google 账户自动同步到不同设备。没有服务器，没有账号系统，隐私完全自主。

## 对比 SlideNote

[SlideNote](https://github.com/maoruibin/SlideNote) 是个不错的类似工具。Slidememos 在基础上增加了：

- Markdown 渲染显示
- 卡片折叠/展开（隐藏敏感信息）
- 文本选中工具栏
- 链接卡片快速打开
