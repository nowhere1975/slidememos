# Slidememos 录屏操作指南

## 一、环境准备清单

### 浏览器设置
- [ ] Chrome 窗口调整为 **1440×900**（DevTools → 设备工具栏选择 Responsive，设 1440×900）
- [ ] 关闭所有无关标签页，只保留扩展页
- [ ] 隐藏书签栏：Chrome 设置 → 外观 → 关闭"显示书签栏"
- [ ] 地址栏输入 `chrome://extensions/`，确认扩展已加载

### 扩展准备
- [ ] 确保 `dist/` 已构建（运行 `npm run build`）
- [ ] 扩展中已有 5-7 条示例数据（用之前 demo-data.json 导入或手动创建）
- [ ] 测试折叠功能正常
- [ ] 测试搜索功能正常

### 系统准备
- [ ] 开启勿扰模式（关闭通知）
- [ ] 桌面壁纸设为纯色（建议 #1a1a2e 或灰色）
- [ ] 清理剪贴板历史
- [ ] 关闭其他应用程序

## 二、分段录制脚本

### 段 1：开场展示（5秒）

**目标**：展示整体界面

**操作步骤**：
1. 确保侧边栏已关闭（点击扩展图标关闭）
2. **【开始录制】**
3. 点击 Chrome 工具栏的 Slidememos 图标
4. 侧边栏滑入，展示 3-4 张卡片
5. **【停顿 2 秒】**
6. **【停止录制】**

**检查要点**：
- 是否展示多种卡片类型（文本、链接）
- 界面是否完整无遮挡
- 动画是否流畅

---

### 段 2：创建 Markdown 笔记（12秒）

**目标**：展示 Markdown 输入和保存

**操作步骤**：
1. **【开始录制】**
2. 点击底部输入框（有明显光标闪烁）
3. **【停顿 0.5 秒】**
4. 输入以下内容（逐行输入，每行后自然停顿）：
   ```
   ## 今日待办

   - [ ] 完成项目文档
   - [x] 代码审查
   - [ ] 发布更新
   ```
5. **【停顿 0.5 秒】**
6. 按 **Ctrl+Enter**（手指可见）
7. 卡片出现在列表顶部
8. **【停顿 2 秒】**
9. **【停止录制】**

**注意事项**：
- 输入不要太快，让观众看清
- Ctrl+Enter 时让键盘入镜
- 保存成功后停顿足够时间

---

### 段 3：创建链接卡片（12秒）

**目标**：展示链接自动识别

**操作步骤**：
1. **【开始录制】**
2. 点击输入框
3. **【停顿 0.5 秒】**
4. 输入：`https://vuejs.org/`
5. **【停顿 0.5 秒】**
6. 按 **Ctrl+Enter**
7. **【停顿 3 秒】**（等待链接解析，标题和图标出现）
8. 鼠标移到新卡片上，展示完整效果
9. **【停顿 1 秒】**
10. **【停止录制】**

**检查要点**：
- 链接解析过程是否可见
- 标题是否成功提取
- 链接图标是否正确显示

---

### 段 4：卡片折叠/展开（10秒）

**目标**：展示折叠动画和隐私保护

**操作步骤**：
1. **【开始录制】**
2. 鼠标移到第一张包含敏感内容的卡片（如 API Key）
3. **【停顿 0.5 秒】**
4. 点击右下角 **向上箭头**（折叠按钮）
5. 等待折叠动画完成
6. **【停顿 2 秒】**
7. 再次点击 **向下箭头**（展开按钮）
8. 等待展开动画完成
9. **【停顿 1 秒】**
10. **【停止录制】**

**注意事项**：
- 选择内容较多的卡片折叠，效果更明显
- 动画过程中不要操作
- 展示折叠后只显示标题

---

### 段 5：Markdown 预览（10秒）

**目标**：展示 Markdown 渲染效果

**操作步骤**：
1. **【开始录制】**
2. 找到之前创建的 Markdown 卡片（今日待办）
3. 点击卡片内容区域进入编辑状态
4. **【停顿 0.5 秒】**
5. 点击右上角 **眼睛图标**（预览按钮）
6. 等待渲染切换
7. **【停顿 2 秒】**
8. 再次点击 **眼睛图标**（显示源码）
9. **【停顿 1 秒】**
10. 点击卡片外部空白处退出编辑
11. **【停顿 1 秒】**
12. **【停止录制】**

**检查要点**：
- 预览模式是否正确渲染
- 切换是否流畅
- 工具栏是否可见

---

### 段 6：链接打开 + 搜索（15秒）

**目标**：展示链接快速打开和实时搜索

**第一部分：链接打开**
1. **【开始录制】**
2. 找到链接卡片（Vue.js）
3. 鼠标缓慢移动到 🔗 图标
4. **【停顿 1 秒】**（展示手型光标和悬停效果）
5. 点击图标
6. **【停顿 1 秒】**（链接在后台打开）

**第二部分：搜索**
7. 点击顶部搜索框
8. **【停顿 0.5 秒】**
9. 输入 `vue`
10. 展示搜索结果（链接卡片保留，其他隐藏）
11. **【停顿 2 秒】**
12. 点击搜索框的 **X** 清除搜索
13. **【停顿 1 秒】**
14. **【停止录制】**

## 三、录制技巧

### 鼠标操作
- **移动速度**：比平时慢 50%，给观众反应时间
- **点击节奏**：点击后停顿 0.3 秒再移动
- **关键位置**：悬停在按钮上时停顿展示

### 键盘操作
- 快捷键**让键盘入镜**（手指可见）
- 复杂输入可以先复制，录屏时粘贴（节省视频时长）

### 停顿节奏
| 场景 | 停顿时间 |
|------|----------|
| 每次点击后 | 0.5-1 秒 |
| 动画播放中 | 等动画完成 + 1 秒 |
| 重要结果展示 | 2 秒 |
| 段落切换 | 2-3 秒（方便后期剪辑） |

### 分段录制原则
- 每段单独录制，方便重录
- 出错就重录该段，不用从头来
- 每段开头结尾多留 1 秒空白

## 四、录屏软件设置

### Screen Studio (Mac) - 推荐
```
分辨率: 1440×900 或 1920×1080
帧率: 30fps
鼠标高亮: 开启
平滑鼠标: 开启（自动优化轨迹）
输出格式: MP4
```

### OBS Studio (Windows/Mac/Linux)
```
来源: 显示器捕获（选择 Chrome 窗口）
分辨率: 1920×1080
帧率: 30
输出格式: MKV（防损坏），后期转 MP4
```

### 快捷键
- **开始/停止录制**: 设为 F8 或 F9（不冲突的键）
- 避免用鼠标点击开始，会录进操作
